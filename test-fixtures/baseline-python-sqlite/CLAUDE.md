## Non-Negotiable Rules (from base framework)

This project inherits every rule from the CW Secure Framework base `CLAUDE.md`:

1. **No secrets in code** — always `os.environ` / `os.Getenv`, always via `make add-secret`
2. **Okta OIDC for auth** — never custom username/password
3. **Parameterized queries** — no string concatenation for SQL
4. **Input validation** — Pydantic strict mode (Python) / struct tags + validator (Go)
5. **Structured logging** — no PII, no tokens, no Authorization headers
6. **Rate limiting** — required on every API
7. **CSRF protection** — state-changing endpoints with cookies
8. **80% test coverage** — enforced by CI, never lowered
9. **Pre-commit hooks** — never bypassed with `--no-verify`

## STRICT MODE — Restricted Data

This project handles **Restricted** or **Highly Restricted** data under CW policy.
Additional rules apply on top of the standard CLAUDE.md:

- **No live data in dev/test/training/demo environments.** All non-prod uses synthetic data.
- **Encryption at rest is mandatory.** `DATABASE_URL` must use TLS (`sslmode=require`).
- **Audit logging is mandatory.** Every auth event, every data access, every export — logged with user_id, timestamp, action.
- **Secret management must use Doppler.** Plain `.env` is blocked by pre-commit.
- **AppSec review is required before first production deploy.** Open a ticket in `#application-security` at least 2 weeks before launch.

## Project-Specific Rules (from wizard)

- **Auth:** Every endpoint except `/healthz` uses `Depends(get_current_user)` (Python) or `middleware.RequireAuth()` (Go). Group claims drive RBAC.
- **Secrets:** Doppler project `test` with configs `dev` / `stg` / `prd`. Local dev uses `.env` (gitignored).
- **Never commit `.env`.** Pre-commit blocks this. No workaround.
- **Database:** SQLite. All queries parameterized. Never `SELECT *`. Migrations in `/migrations`, numbered sequentially.
- **SQLite warning:** Single-writer. Not for multi-user production. Migrate before scale.
- **API shape:** REST. Resource-based URLs. Standard HTTP verbs. Response format: flat JSON, no wrapper.
- **Traffic:** Spiky. Queue buffers must cap. Circuit breakers on every external call. No unbounded fan-out.
