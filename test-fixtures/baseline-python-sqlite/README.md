## Architecture

FastAPI for requests, Celery/RQ for async jobs, Redis as broker.

Stack layers (top to bottom):
```
HTTP request
  ↓ middleware (auth · rate limit · request ID · headers)
  ↓ routes/   — thin handlers (10–20 lines)
  ↓ services/ — business logic
  ↓ repositories/ — parameterized queries
  ↓ SQLite
```
